Class: kb.DefaultObservable

Defined in: src/defaults/default-observable.coffee

Overview

Used to provide a default value when an observable is null, undefined, or the empty string.

Examples:

Provide a observable with observable and/or non observable default argument in the form of: 

var wrapped_name = kb.defaultObservable(kb.observable(model, 'name'), '(no name)');

Instance Method Summary

  • # (void) destroy() Required clean up function to break cycles, release view models, etc.
  • # (void) setToDefault() Forces the observable to take the default value.
